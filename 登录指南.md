# 🔐 Gojica 登录指南

## 登录方式

Gojica 提供 **3种登录方式**：

### 1. 🧪 开发模式登录（推荐测试使用）

**用途**: 开发测试阶段使用，无需真实微信账号

**步骤**:
1. 打开前端页面 → 登录页
2. 输入任意手机号（格式正确即可）
3. 点击 **"开发模式登录(调试)"** 按钮
4. 自动登录成功，跳转到首页

**API接口**:
```
POST /api/v1/auth/dev-login
```

**特点**:
- ✅ 无需验证码
- ✅ 无需微信授权
- ✅ 快速测试
- ⚠️ 仅开发环境可用

---

### 2. 📱 微信一键登录

**用途**: 正式环境使用

**步骤**:
1. 点击 **"微信一键登录"** 按钮
2. 微信授权弹窗
3. 授权后自动登录

**API接口**:
```
POST /api/v1/auth/login
参数:
{
  "code": "微信授权code",
  "identity": "fan/musician/band/venue"
}
```

**特点**:
- ✅ 真实微信账号
- ✅ 安全性高
- ⚠️ 需要真实手机号
- ⚠️ 开发环境可能不可用

---

### 3. 📝 手机号 + 验证码登录

**用途**: 未来扩展功能

**步骤**:
1. 输入手机号
2. 点击 **"获取验证码"**
3. 输入验证码（6位数字）
4. 点击登录

**API接口**:
```
POST /api/v1/auth/login
参数:
{
  "phone": "13800138000",
  "code": "123456",
  "identity": "fan"
}
```

**特点**:
- ✅ 传统登录方式
- ✅ 用户熟悉
- ⏳ 暂未完全实现

## 📝 开发模式登录详解

### 为什么推荐开发模式登录？

1. **快速**: 无需任何验证，即点即用
2. **便捷**: 不需要真实手机号
3. **安全**: 仅开发环境可用，不会影响生产
4. **灵活**: 可切换身份（fan/musician/band/venue）

### 身份类型

| 身份 | 说明 | 英文 |
|------|------|------|
| 粉丝 | 普通用户 | `fan` |
| 音乐人 | 音乐爱好者/乐手 | `musician` |
| 乐队 | 乐队管理员 | `band` |
| 场地 | 排练室/演出场地 | `venue` |

### 代码实现

**前端登录代码**:
```javascript
// 调用 API
const res = await authApi.login(phone, 'fan')

// 成功响应
if (res.code === 1000 || res.code === 1000) {
  // 保存 token
  api.default.setToken(res.data.accessToken)
  
  // 保存用户信息
  uni.setStorageSync('userInfo', res.data.user)
  
  // 跳转到首页
  uni.switchTab({ url: '/pages/home/index' })
}
```

**后端开发模式登录**:
```javascript
// server/src/services/authService.js
const devLogin = (identity = 'fan') => {
  // 在开发环境，随机创建一个测试用户
  const user = createTestUser(identity)
  
  // 生成 JWT token
  const token = jwt.sign({ userId: user.id }, SECRET, { expiresIn: '24h' })
  
  return { token, userInfo: user }
}
```

## 🔍 登录状态检查

### 前端检查

打开浏览器控制台（F12）:

```javascript
// 检查是否已登录
uni.getStorageSync('userInfo')
// 应该返回用户信息对象

// 检查 token
uni.getStorageSync('token')  
// 应该返回 JWT token 字符串
```

### 后端检查

```bash
curl http://localhost:3000/api/v1/users/profile
```

如果没有登录，会返回:
```json
{
  "code": 401,
  "message": "未授权"
}
```

登录后会返回用户信息。

## 🐛 常见登录问题

### 1. 登录后不跳转
**原因**: API 调用失败
**解决**: 
- 检查后端是否运行
- 检查控制台错误信息
- 查看网络请求

### 2. token 过期
**原因**: JWT token 过期（默认24小时）
**解决**: 
- 重新登录
- 或调用刷新 token 接口

### 3. 微信登录不可用
**原因**: 开发环境未配置微信 AppID
**解决**: 
- 使用开发模式登录
- 或配置微信开发者账号

### 4. 验证码发送失败
**原因**: 短信服务未配置
**解决**: 
- 使用开发模式登录
- 或配置短信服务商
