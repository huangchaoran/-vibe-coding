## 全局设计系统

### 配色方案

| 用途 | 色值 | 示例 |
|------|------|------|
| 主色/强调 | `#e94560` | 按钮、选中态、标题高亮 |
| 深色背景 | `#0f0f23` / `#0d0d1f` | 页面底色 |
| 卡片背景 | `#16213e` | 卡片、输入框 |
| 边框 | `#1a1a3e` / `#1a1a4e` | 分割线、边框 |
| 次要文字 | `#5a5a8a` | 图标、未选中文字 |
| 主文字 | `#ffffff` | 标题、重要信息 |
| 辅助文字 | `#a0a0c0` | 副标题、描述 |
| 成功绿 | `#27ae60` | 进行中状态 |

### 字体规范

- 标题: 16-20px, 加粗 `#fff`
- 正文: 13-14px, `#c0c0e0`
- 辅助: 11-12px, `#a0a0c0`
- 价格: 14-20px, 加粗 `#e94560`

### 间距系统

- 页面边距: 14px
- 卡片间距: 12px
- 组件内边距: 12-16px
- 圆角: 8-14px (小元素), 20px (按钮)

  id: string,

### 交互行为

| 元素 | 交互 |
|------|------|
| 搜索栏 | 点击 → 跳转搜索页 |
| 轮播图 | 左右滑动切换，自动轮播3秒 |
| 统计数字 | 点击 → 跳转对应分类 |
| 乐队卡片 | 点击关注 → 切换状态，发送请求 |
| 活动卡片 | 点击报名 → 跳转活动详情或直接报名 |
| 底部导航 | 切换 tab，高亮当前页 |

## 总结

Gojica 音乐社交平台小程序原型图包含以下核心功能：

1. **认证模块** - 微信一键登录、手机号登录、开发调试模式
2. **首页模块** - 轮播图、统计数据、热门乐队、近期活动
3. **乐队模块** - 乐队列表、详情、创建、管理
4. **活动模块** - 活动列表、详情、报名、收藏
5. **招募模块** - 职位招募、申请加入
6. **商品模块** - 二手乐器交易、收藏、购买
7. **动态模块** - 帖子发布、互动（点赞、评论、分享）
8. **个人中心** - 资料编辑、身份管理、订单、预约、设置
9. **搜索模块** - 全局搜索、多类型结果筛选
10. **排练室模块** - 排练室预约
