## 🚀 运行测试

### 方法 1：快速截图测试（推荐新手）

打开一个新的命令行窗口，运行：

```bash
cd "c:\Users\Administrator\Desktop\千早爱音给我助教\Gojica2.0\Gojica2.0前端"
node run-test.js
```

这会：
- 启动 Chrome 浏览器
- 打开你的应用
- 截图保存到 `test-results/` 目录
- 保持浏览器打开让你查看

### 方法 2：完整功能测试

```bash
cd "c:\Users\Administrator\Desktop\千早爱音给我助教\Gojica2.0\Gojica2.0前端"
node playwright-test.js
```

这会执行完整的测试流程：
- 登录页面测试
- 表单填写
- TabBar 导航测试
- 多页面截图

## 📸 查看测试结果

所有截图保存在：`test-results/` 目录

## 🔧 自定义测试

编辑 `playwright-test.js` 文件来修改测试步骤。

## 💡 提示

- 如果 Playwright 未安装，先运行：`npm install playwright`
- 如果浏览器报错，运行：`npx playwright install chromium`
- 保持 HBuilderX 中的项目运行不要关闭
