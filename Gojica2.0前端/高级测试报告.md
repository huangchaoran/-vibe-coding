## ⚠️ 发现的问题

### 问题：收藏功能数据库错误
**严重程度**: 中等（不影响核心功能）

**错误信息**:
```
Unknown column 'target_type' in 'where clause'
Load bands error: Error: Unknown column 'target_type'
```

**原因**: bands 表查询时使用了 `target_type` 字段，但该字段在数据库中不存在

**影响范围**: 乐队列表加载时可能报错

**建议修复方案**:

```sql
-- 方案1: 检查并添加 target_type 字段
ALTER TABLE bands ADD COLUMN target_type VARCHAR(50) DEFAULT 'band';

-- 方案2: 检查后端查询代码
-- 查看 src/services/bandService.js 中的查询语句
-- 可能需要移除 target_type 条件或添加该字段
```
