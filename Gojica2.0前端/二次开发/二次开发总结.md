## 📋 一、项目概述

### 1.1 项目定位
**Gojica - 音乐人社区与服务平台**

一个服务于乐队、乐手及音乐爱好者的垂直社区平台，整合以下四大核心功能：
- 🎵 **乐队管理** - 乐队入驻、成员管理、作品展示
- 📅 **活动管理** - 活动发布、在线报名、签到核销
- 🏠 **排练房预约** - 场地展示、时段预约、订单管理
- 🛒 **二手设备交易** - 商品发布、咨询聊天、担保交易

### 1.2 附加功能
- 🎸 **乐手招募** - 乐队发布招募、用户投递申请
- 📱 **社区动态** - 图文动态发布、点赞评论
- 💬 **即时咨询** - 商品咨询聊天
- 🔔 **通知系统** - 多类型通知推送

### 1.3 技术架构

| 层级 | 技术栈 | 状态 |
|------|--------|------|
| **小程序端** | UniApp (Vue3 + TypeScript) | ⚠️ 部分完成 |
| **服务端** | Node.js + Express + MySQL 8.0 | ✅ 完成 |
| **管理后台** | Vue3 + Element Plus + ECharts | ✅ 完成 |
| **认证系统** | JWT + 微信授权 | ✅ 完成 |
