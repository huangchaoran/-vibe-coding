# Gojica 音乐社交平台 - 页面描述文档

> 技术栈：UniApp (Vue3) + 微信小程序
> 设计风格：深色音乐主题（黑底 #0d0d1f + 玫瑰红 #e94560）

---

## 二、页面清单

### 1. 登录页
**路径**: `pages/login/index`

**布局**:
- 垂直居中布局
- 顶部：大 Logo 图标（🎸）+ 应用名 "GOJICA" +  slogan "发现音乐·连接人脉·玩转乐圈"
- 中部：手机号输入框 + 验证码输入框
- 底部：微信一键登录按钮（绿色 #07c160）+ "或" 分隔线 + 开发模式登录按钮

**交互**:
- 点击微信登录调起微信授权
- 开发模式可跳过授权直接登录

---

### 2. 首页
**路径**: `pages/index/index`

**顶部导航栏**:
- 左侧：Logo "🎸 Gojica"
- 右侧：通知图标 + 搜索图标

**内容区域**（从上至下）:
1. **搜索栏**: 圆角输入框，placeholder "搜索乐队、活动、商品..."
2. **轮播图**: 高度 300rpx，自动轮播，显示活动推荐
   - 标题 + 副标题 + 右上角标签 "🎵 活动推荐"
3. **统计卡片行**: 4列等宽卡片
   - 2.8k 乐队 | 1.2k 活动 | 8.6k 音乐人 | 320 招募中
4. **热门乐队**: 横向滚动列表
   - 每个卡片：头像(渐变色背景+emoji) + 乐队名 + 风格·人数 + 关注按钮
5. **近期活动**: 纵向卡片列表
   - 封面图(渐变色+emoji+状态标签) + 标题 + 日期·地点 + 价格·报名人数 + 报名按钮

---

### 3. 乐队列表页
**路径**: `pages/band/list`

**顶部**:
- 标题 "🎸 乐队" + 搜索图标 + 创建按钮(+)

**风格筛选栏**:
- 横向滚动标签：全部 | Rock | Jazz | Pop | Folk | Metal | Blues
- 选中态：玫瑰红背景

**列表项**:
- 左侧：方形头像(渐变色+emoji)
- 中间：乐队名 + 风格·成员数·关注人数
- 右侧：风格标签(带边框)

---

### 4. 乐队详情页
**路径**: `pages/band/detail`

**顶部封面**:
- 全宽渐变色背景 + 大 emoji
- 左上角返回按钮(圆形半透明)

**信息区**:
- 乐队名(大标题)
- 标签行：风格 | 人数 | 成立年份
- 简介文字

**成员区**:
- 标题 "🎤 乐队成员"
- 横向排列：头像(圆形) + 姓名 + 角色(玫瑰红色)

**活动区**:
- 标题 "🎉 参与活动"
- 活动卡片：标题 + 日期·地点

**底部固定操作栏**:
- 左侧：💌 私信（次要按钮，描边样式）
- 右侧：❤️ 关注/已关注（主按钮，填充样式）

---

### 5. 创建乐队页
**路径**: `pages/band/create`

**头像上传区**:
- 圆形虚线边框上传框 + "上传乐队头像"提示

**表单区**:
1. 乐队名称 *（输入框）
2. 音乐风格 *（标签选择：Rock/Jazz/Pop/Folk/Metal/Blues）
3. 乐队简介（多行文本框）
4. 所在城市（picker选择器）

**底部**: 🎸 创建乐队（主按钮）

**底部操作栏**: 💌 收藏 | 🎟 立即报名

## 三、通用组件规范

### 按钮样式
| 类型 | 样式 |
|------|------|
| 主按钮 | 玫瑰红背景 #e94560，白色文字，圆角 40rpx |
| 次要按钮 | 透明背景，玫瑰红边框+文字 |
| 禁用按钮 | 灰色背景 #333，灰色文字 |

### 卡片样式
- 背景：#16213e
- 圆角：24rpx-28rpx
- 边框：1rpx solid #1a1a4e

### 头像规格
| 类型 | 尺寸 | 圆角 |
|------|------|------|
| 大头像 | 140rpx | 圆形 |
| 中头像 | 92rpx | 20rpx（方形）或圆形 |
| 小头像 | 80rpx | 圆形 |
| 列表头像 | 80rpx | 圆形 |
