#### 预期菜单项
1. 仪表盘
2. 用户管理
3. 内容审核
4. 订单管理
5. 轮播图管理
6. 数据统计

#### 测试命令
```javascript
// 检查菜单项数量
const menuItems = await page.$$('.el-menu-item');
console.log(`菜单项数量: ${menuItems.length}`);

// 检查菜单文本
const menuTexts = await page.evaluate(() => {
  const items = document.querySelectorAll('.el-menu-item');
  return Array.from(items).map(item => item.innerText.trim());
});
console.log('菜单项:', menuTexts);
```

- [ ] 最小字符数限制

```javascript

const inputs = await page.$$('input');
console.log(`输入框数量: ${inputs.length}`);

const inputInfo = await page.evaluate(() => {

  return Array.from(inputs).map(input => ({
```

