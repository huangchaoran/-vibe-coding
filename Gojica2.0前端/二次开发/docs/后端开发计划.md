## 一、项目概述

### 1.1 项目背景

Gojica 是一个服务于乐队、乐手及音乐爱好者的垂直社区平台，需要开发完整的后端服务系统。根据第一次开发的经验教训，本次开发将重点关注：

- 严格按照接口规范文档开发
- 完善的错误处理和日志系统
- 充分的测试覆盖
- 清晰的代码结构

### 1.2 技术栈

```
运行时: Node.js 18 LTS
框架: Express 4.x
数据库: MySQL 8.0
ORM: Sequelize 6.x
认证: JWT (jsonwebtoken)
密码加密: bcryptjs
文件上传: multer
验证: express-validator
限流: express-rate-limit
日志: winston
跨域: cors
环境变量: dotenv
测试: Jest + Supertest
```

