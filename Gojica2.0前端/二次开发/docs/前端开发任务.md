## 一、项目概述

### 1.1 项目背景

Gojica 是一个面向音乐人的社区与服务平台，旨在连接音乐人、乐队、场地和乐迷，提供活动组织、排练房预约、商品交易等功能。

### 1.2 项目目标

1. **小程序客户端**: 为普通用户提供便捷的移动端体验，包括浏览乐队、参加活动、预约排练房、购买商品等功能
2. **后台管理系统**: 为平台管理员提供用户管理、内容审核、数据统计等功能

## 三、小程序客户端功能需求

### 3.1 用户模块

#### 3.1.1 微信登录
- **功能描述**: 使用微信授权登录
- **接口对接**:
  - `POST /api/v1/auth/login` - 微信登录
  - `POST /api/v1/auth/logout` - 退出登录
- **页面流程**:
  ```
  启动小程序 → 检测登录状态
      ↓
  已登录 → 首页
      ↓
  未登录 → 微信授权 → 绑定身份 → 首页
  ```
- **交互要求**:
  - 自动获取微信昵称和头像
  - 首次登录需选择身份（粉丝/音乐人/乐队/场地）
  - 保存登录态（7天有效期）

#### 3.1.2 用户资料
- **功能描述**: 查看和编辑个人资料
- **接口对接**:
  - `GET /api/v1/auth/profile` - 获取个人信息
  - `PUT /api/v1/auth/profile` - 更新个人信息
  - `POST /api/v1/auth/bind-identity` - 绑定身份
- **页面元素**:
  - 头像（支持点击更换）
  - 昵称
  - 手机号（脱敏显示：138****8000）
  - 身份标识
  - 个人简介
  - 音乐偏好标签
- **交互要求**:
  - 修改头像调用微信选择图片API
  - 修改手机号需短信验证
  - 身份绑定后不可更改

#### 3.1.3 我的页面
- **功能描述**: 个人中心聚合页
- **页面元素**:
  - 用户信息卡片
  - 统计数据（关注数、粉丝数、收藏数）
  - 功能入口列表
    - 我的收藏
    - 我参加的活动
    - 我的预约
    - 我的订单
    - 我的发布
    - 设置

## 五、网络请求封装

### 5.1 小程序端请求封装（UniApp）

```typescript
// utils/request.ts
import { BASE_URL, TIMEOUT } from '@/config';

// 请求配置接口
interface RequestConfig {
  url: string;
  method?: 'GET' | 'POST' | 'PUT' | 'DELETE';
  data?: any;
  header?: Record<string, string>;
  loading?: boolean;
  loadingText?: string;
}

// 响应接口
interface ResponseData<T = any> {
  code: number;
  message: string;
  data: T;
  timestamp: number;
}

// Token 获取函数
function getToken(): string {
  return uni.getStorageSync('token') || '';
}

// 响应码判断
function isSuccess(code: number): boolean {
  return code === 200 || (code >= 1000 && code < 2000);
}

// 请求封装
export function request<T = any>(config: RequestConfig): Promise<T> {
  const { url, method = 'GET', data, header = {}, loading = true, loadingText = '加载中...' } = config;

  // 显示加载提示
  if (loading) {
    uni.showLoading({ title: loadingText, mask: true });
  }

  return new Promise((resolve, reject) => {
    uni.request({
      url: BASE_URL + url,
      method,
      data,
      header: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${getToken()}`,
        'X-Requested-With': 'XMLHttpRequest',
        ...header
      },
      timeout: TIMEOUT,
      success: (res) => {
        if (loading) {
          uni.hideLoading();
        }

        if (res.statusCode === 200) {
          const response = res.data as ResponseData<T>;
          
          if (isSuccess(response.code)) {
            resolve(response.data);
          } else if (response.code === 2005 || response.code === 2003 || response.code === 2004) {
            // Token 过期，跳转登录
            uni.removeStorageSync('token');
            uni.removeStorageSync('userInfo');
            uni.reLaunch({ url: '/pages/login/login' });
            reject(new Error(response.message));
          } else {
            uni.showToast({
              title: response.message,
              icon: 'none'
            });
            reject(new Error(response.message));
          }
        } else if (res.statusCode === 401) {
          uni.removeStorageSync('token');
          uni.reLaunch({ url: '/pages/login/login' });
          reject(new Error('未授权'));
        } else {
          uni.showToast({
            title: '网络请求失败',
            icon: 'none'
          });
          reject(new Error(`HTTP ${res.statusCode}`));
        }
      },
      fail: (err) => {
        if (loading) {
          uni.hideLoading();
        }
        uni.showToast({
          title: '网络连接失败',
          icon: 'none'
        });
        reject(err);
      }
    });
  });
}

// 快捷方法封装
export const get = <T = any>(url: string, data?: any, loading = true) => 
  request<T>({ url, method: 'GET', data, loading });

export const post = <T = any>(url: string, data?: any, loading = true) => 
  request<T>({ url, method: 'POST', data, loading });

export const put = <T = any>(url: string, data?: any, loading = true) => 
  request<T>({ url, method: 'PUT', data, loading });

export const del = <T = any>(url: string, data?: any, loading = true) => 
  request<T>({ url, method: 'DELETE', data, loading });
```

### 5.2 后台管理系统请求封装（Axios）

```typescript
// utils/request.ts
import axios, { AxiosInstance, AxiosRequestConfig, AxiosResponse } from 'axios';
import { ElMessage } from 'element-plus';
import router from '@/router';

// 创建实例
const service: AxiosInstance = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 30000,
  headers: {
    'Content-Type': 'application/json'
  }
});

// 请求拦截器
service.interceptors.request.use(
  (config) => {
    const token = localStorage.getItem('adminToken');
    if (token) {
      config.headers.Authorization = `Bearer ${token}`;
    }
    return config;
  },
  (error) => {
    return Promise.reject(error);
  }
);

// 响应拦截器
service.interceptors.response.use(
  (response: AxiosResponse) => {
    const { code, message, data } = response.data;

    if (code === 200 || (code >= 1000 && code < 2000)) {
      return data;
    }

    if (code === 2005 || code === 2003 || code === 2004) {
      // Token 过期
      localStorage.removeItem('adminToken');
      router.push('/login');
      ElMessage.error('登录已过期，请重新登录');
      return Promise.reject(new Error(message));
    }

    ElMessage.error(message);
    return Promise.reject(new Error(message));
  },
  (error) => {
    if (error.response) {
      const { status, data } = error.response;
      
      switch (status) {
        case 401:
          localStorage.removeItem('adminToken');
          router.push('/login');
          ElMessage.error('未授权，请重新登录');
          break;
        case 403:
          ElMessage.error('无权限访问');
          break;
        case 404:
          ElMessage.error('请求资源不存在');
          break;
        case 500:
          ElMessage.error('服务器错误');
          break;
        default:
          ElMessage.error(data.message || '请求失败');
      }
    } else {
      ElMessage.error('网络连接失败');
    }
    return Promise.reject(error);
  }
);

// 请求方法
export const request = {
  get: <T = any>(url: string, params?: any, config?: AxiosRequestConfig) =>
    service.get<T>(url, { params, ...config }),
  
  post: <T = any>(url: string, data?: any, config?: AxiosRequestConfig) =>
    service.post<T>(url, data, config),
  
  put: <T = any>(url: string, data?: any, config?: AxiosRequestConfig) =>
    service.put<T>(url, data, config),
  
  delete: <T = any>(url: string, params?: any, config?: AxiosRequestConfig) =>
    service.delete<T>(url, { params, ...config }),
  
  upload: <T = any>(url: string, file: File, data?: Record<string, any>) => {
    const formData = new FormData();
    formData.append('file', file);
    if (data) {
      Object.entries(data).forEach(([key, value]) => {
        formData.append(key, value as string);
      });
    }
    return service.post<T>(url, formData, {
      headers: { 'Content-Type': 'multipart/form-data' }
    });
  }
};

export default service;
```

## 十二、注意事项

### 12.1 开发规范

1. **代码规范**: 严格遵循 ESLint + Prettier 配置
2. **命名规范**: 使用有意义的命名，统一命名风格
3. **注释规范**: 复杂逻辑需添加注释
4. **Git 规范**: 分支命名、提交信息规范化

### 12.2 安全注意

1. **敏感信息**: 不在前端存储敏感信息
2. **Token 管理**: 安全存储和传输
3. **用户输入**: 做好前端验证，不依赖后端验证
4. **XSS 防护**: 避免使用 v-html 处理用户输入

### 12.3 性能注意

1. **图片优化**: 使用懒加载、压缩、WebP
2. **列表优化**: 使用虚拟列表、长列表优化
3. **缓存策略**: 合理使用本地缓存
4. **代码分割**: 按需加载页面和组件
