# Gojica 项目测试指南

## 方案一：使用 Playwright 测试 H5 版本（推荐用于开发调试）

### 步骤 1：编译成 H5 版本

在 HBuilderX 中：
1. 打开项目
2. 点击 **运行** → **运行到浏览器** → **Chrome**
3. 或者点击 **发行** → **网站-PC-Web** → **网站（仅H5）**

### 步骤 2：运行测试脚本

```bash
# 安装 Playwright（如果还没有）
npm install playwright

# 安装浏览器依赖
npx playwright install chromium

# 运行测试
node playwright-test.js
```

## 方案二：使用微信开发者工具测试小程序版本

### 步骤 1：打开微信开发者工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具
3. 导入项目：`项目目录 → Gojica2.0前端/unpackage/dist/dev/mp-weixin`
4. AppID：可以先选择"测试号"

### 步骤 2：测试功能

在微信开发者工具中可以：
- 预览小程序
- 使用模拟器测试不同设备
- 查看控制台日志
- 调试 WXML/CSS/JS

## 方案三：自动化测试微信小程序

微信小程序不支持 Playwright，但可以使用：

### 工具 1：Miniprogram Automator
```bash
npm install miniprogram-automator
```

### 工具 2：jest-miniprogram
针对小程序组件的测试框架

## 当前测试脚本说明

`playwright-test.js` 功能：
- ✅ 启动本地 Chrome
- ✅ 访问登录页面
- ✅ 截图保存到 `test-results/` 目录
- ✅ 填写表单
- ✅ 点击按钮
- ✅ 导航到其他页面
- ✅ 测试 TabBar 切换

## 快速开始

1. **编译 H5 版本**
   - HBuilderX → 运行 → 运行到浏览器 → Chrome

2. **安装 Playwright**（新开终端）
   ```bash
   cd Gojica2.0前端
   npm install playwright
   npx playwright install chromium
   ```

3. **运行测试**（在新终端）
   ```bash
   node playwright-test.js
   ```

4. **查看测试结果**
   - 截图保存在：`test-results/` 目录
   - 观察浏览器窗口操作
