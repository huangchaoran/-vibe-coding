### 问题 2: 搜索功能 URL 编码问题

**症状**: `Request path contains unescaped characters`

**原因**: 中文关键词 `q=摇滚` 没有 URL 编码

**当前代码**:
```javascript
path: '/api/v1/search?q=摇滚&type=band'
```

**修复方案**:
```javascript
// 方案 1: URL 编码
path: '/api/v1/search?q=' + encodeURIComponent('摇滚') + '&type=band'

// 方案 2: 使用 query string
const params = new URLSearchParams({ q: '摇滚', type: 'band' })
path: '/api/v1/search?' + params.toString()
```

**前端代码** (`api.uts`):
```typescript
export const searchApi = {
  search: (params) => apiGet('/search?' + new URLSearchParams(params).toString())
}
```

### 问题 4: Sequelize 内部字段暴露

**症状**: 动态列表返回包含 Sequelize 内部字段

**问题数据**:
```json
{
  "author": {
    "PreviousDataValues": {...},
    "uniqno": 1,
    "Changed": {},
    "Options": {...},
    "isNewRecord": false
  }
}
```

**原因**: `deepConvert` 函数未正确处理嵌套的 Sequelize 实例

**修复方案**:

```javascript
// src/utils/response.js
const deepConvert = (obj, seen = new WeakSet()) => {
  if (isPrimitive(obj)) return obj;
  if (seen.has(obj)) return '[Circular]';
  if (Array.isArray(obj)) {
    seen.add(obj);
    return obj.map(item => deepConvert(item, seen));
  }
  
  if (typeof obj === 'object' && obj !== null) {
    // ✅ 检查是否是 Sequelize 实例
    if (typeof obj.toJSON === 'function') {
      return deepConvert(obj.toJSON(), seen);
    }
    
    // ✅ 排除 Sequelize 内部字段
    const excludeKeys = [
      'PreviousDataValues', 'uniqno', 'Changed', 'Options',
      'isNewRecord', 'dataValues', 'parent', 'include',
      '_previousDataValues', '_hasWarnings', '_created',
      '_updated', '_modelOptions', '_options', '_values'
    ];
    
    seen.add(obj);
    const result = {};
    const keys = Object.keys(obj);
    
    for (const key of keys) {
      if (excludeKeys.includes(key)) continue;
      const camelKey = snakeToCamel(key);
      result[camelKey] = deepConvert(obj[key], seen);
    }
    return result;
  }
  
  return obj;
};
```

## 🟡 潜在问题 (需进一步验证)

### 问题 5: 前端 Token 存储字段名

**前端代码** (`pages/login/index.uvue`):
```typescript
const res = await authApi.login(phone.value, 'fan')
api.default.setToken(res.data?.accessToken)  // ❌ 应该是 accessToken
uni.setStorageSync('userInfo', res.data?.user || {})  // ❌ 应该是 user
```

**后端返回**:
```json
{
  "code": 1000,
  "data": {
    "token": "...",
    "userInfo": {...}
  }
}
```

**问题**:
- 后端返回 `token` 但前端读取 `accessToken`
- 后端返回 `userInfo` 但前端读取 `user`

**修复方案**:

#### 方案 A: 修改前端 (推荐)

```typescript
// pages/login/index.uvue
const devLogin = async () => {
  try {
    const res = await authApi.login(phone.value, 'fan')
    if (res && (res.code === 2000 || res.code === 1000)) {
      // ✅ 修改字段名
      api.default.setToken(res.data?.token)  // accessToken -> token
      uni.setStorageSync('userInfo', res.data?.userInfo || {})  // user -> userInfo
      uni.switchTab({ url: '/pages/home/index' })
    }
  } catch (e) {
    // ...
  }
}
```

#### 方案 B: 修改后端

```javascript
// src/controllers/authController.js
ApiResponse.success(res, {
  accessToken: result.token,  // token -> accessToken
  refreshToken: result.refreshToken,
  expiresIn: 86400,
  user: result.userInfo,  // userInfo -> user
}, '登录成功');
```
