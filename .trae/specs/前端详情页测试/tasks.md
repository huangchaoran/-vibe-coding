# Tasks: 前端详情页测试

## 任务概述
对前端5个详情页面进行功能测试，验证API对接、页面展示、用户交互是否正常

---

## 任务清单

- [ ] Task 1: 准备测试数据
  - 在后端数据库中创建测试乐队数据（包含成员）
  - 创建测试活动数据
  - 创建测试商品数据
  - 创建测试帖子数据（含评论）
  - 创建测试招募数据

- [ ] Task 2: 测试乐队详情页
  - 访问乐队详情页，验证页面加载
  - 检查乐队基本信息展示（名称、风格、简介、封面图）
  - 检查成员列表展示
  - 测试关注/取消关注按钮功能
  - 检查近期活动列表
  - 测试返回按钮功能

- [ ] Task 3: 测试活动详情页
  - 访问活动详情页，验证页面加载
  - 检查活动基本信息展示
  - 检查时间地点信息展示
  - 检查报名人数进度条
  - 测试报名/取消报名功能
  - 检查发布者/乐队信息

- [ ] Task 4: 测试商品详情页
  - 访问商品详情页，验证页面加载
  - 检查商品图片展示
  - 检查价格信息（原价格、现价格）
  - 检查商品描述展示
  - 检查卖家/乐队信息
  - 测试收藏功能

- [ ] Task 5: 测试帖子详情页
  - 访问帖子详情页，验证页面加载
  - 检查帖子内容展示（文字、图片）
  - 检查作者信息展示
  - 测试点赞/取消点赞功能
  - 检查评论列表展示
  - 测试评论输入和提交功能

- [ ] Task 6: 测试招募详情页
  - 访问招募详情页，验证页面加载
  - 检查招募标题和描述展示
  - 检查乐器要求列表
  - 检查发布方信息展示
  - 测试申请功能

## 任务依赖
- Task 1 应在其他测试前完成
- Task 2-6 可并行执行

## 验证方式
- 页面正常加载，无报错
- 数据正确展示
- 交互功能正常响应
- API调用无错误
